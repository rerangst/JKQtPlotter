JKQTPXYParametrizedScatterGraph Class Reference
JKQTPlotter: Qt based Scientific Plotter Class » Plot Elements » Graph Classes » Line/Symbol Graphs » Parametrized Line/Scatter Graphs

This implements xy scatter plots (like JKQTPXYLineGraph), but the color and size of the symbols may be taken from a column.set the properties sizeColumn and/or colorColumn to change the size and/or color of the symbols according to the values in the column. More...

Detailed Description

This implements xy scatter plots (like JKQTPXYLineGraph), but the color and size of the symbols may be taken from a column.

set the properties sizeColumn and/or colorColumn to change the size and/or color of the symbols according to the values in the column.

Different Styles of Parametrized Scatter/Line Graphs
JKQTPXYParametrizedScatterGraph with symbols organized in a grid
See also
JKQTPXYParametrizedErrorScatterGraph, Example (JKQTPlotter): Scatter Graph with Parametrized Symbols/Colors , Example (JKQTPlotter): Draw an Artistic Image with a Parametrized Scatter Graph
