37 /** \brief this virtual base class of every element, which is part of a JKQTPlotter plot and may appear in its key
38  * (basically any type of graph, except overlay elements!)
39  * \ingroup jkqtplotter_basegraphs
40  *
41  * Each possible graph is represented by a child of this class. So additional plots may be created by
42  * deriving new JKQTPGraph classes. To do so implement/overwrite these functions:
43  * - void draw(JKQTPEnhancedPainter& painter);
44  * - void drawKeyMarker(JKQTPEnhancedPainter& painter, QRectF& rect);
45  * - bool getXMinMax(double& minx, double& maxx, double& smallestGreaterZero);
46  * - bool getYMinMax(double& miny, double& maxy, double& smallestGreaterZero);
47  * - QColor getKeyLabelColor()=0;
48  * .
49  *
50  * Optionally you may also overwrite these functions to draw elements outside the actual plot area (like e.g. colorbars):
51  * - void getOutsideSize(JKQTPEnhancedPainter& painter, int& leftSpace, int& rightSpace, int& topSpace, int& bottomSpace);
52  * - void drawOutside(JKQTPEnhancedPainter& painter, QRect leftSpace, QRect rightSpace, QRect topSpace, QRect bottomSpace);
53  * .
54  *
55  * In addition this class provudes protected
56  * functions that do coordinate transforms based on the current coordinate system, of the paren
57  * JKQTPlotter (i.e. using the axes JKQTPLott:xAxis and JKQTPlotter::yAxis as basis for the plotting).
58  *
59  * \see \ref jkqtplotter_graphsgroup_classstructure
60  */

184 /** \brief this virtual base class of the (data-column based) graphs,
185  * which are part of a JKQTPlotter plot and which use the coordinate system
186  * of the JKQTPlotter (i.e. the two coordinate axes getXAxis() and getYAxis())
187  * as basis for the graphs
188  * \ingroup jkqtplotter_basegraphs
189  *
190  * This class adds features to work with data columns.
191  * - There are two properties datarange_start and datarange_end. By default they are -1 and therefore ignored.
192  * if they are != -1 the plotter only displays the datapoints with the indexes [datarange_start .. datarange_end]
193  * although there might be more data points available (range [0 .. maxDataPoints]). The datarange is cut at the
194  * full range, i.e. if datarange_end>maxDataPoints the plotter displays [datarange_start .. maxDataPoints].
195  * - Also there is a virtual function usesColumn() which checks whether a given column is used by this graph.
196  * Override this function in your derived graphs to indicate to JKQTPlotter / JKQTBasePlotter , which columns
197  * from the internal JKQTPDatastore are actually used. This information can be used e.g. for graph-specific data-export.
198  * .
199  *
200  * \see \ref jkqtplotter_graphsgroup_classstructure
201  */
