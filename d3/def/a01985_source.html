38 /*! \brief class for a contour plot
39  * \ingroup jkqtplotter_imagelots_contour
40  * calculates the contour of a given image using the CONREC algorithm
41  * <a href="http://paulbourke.net/papers/conrec/">http://paulbourke.net/papers/conrec/</a>
42  * The implementation for Qt is adapted from Qwt
43  * <a href="http://qwt.sourceforge.net/">http://qwt.sourceforge.net/</a>
44  *
45  * The contour lines are calculated only once and then redrawn to save plotting time.
46  * Make sure you add a new graph when your data changes. The contour levels can be defined in
47  * contourLevels, each elemt in the list represents a contour plane. You can also create contour
48  * levels with createContourLevels(). The levels are linearly spaced between the maximum and minimum
49  * value in your data. For logarithmic data, use createContourLevelsLog() to create contour levels
50  * with logarithmic spacing.
51  *
52  * \author Sebastian Isbaner, 2013-2014
53  *
54  * \image html JKQTPContour.png
55  */
