35 /*! \brief Descendents of this class add data fields for error indicators and methods to plot them to a class.
36  \ingroup jkqtplotter_basegraphserrors
37 
38  This class is meant to be used with multiple inheritance. I.e. if you implemented some kind of plot
39  you may derive an error plot class in which you will have to overwrite the JKQTPGraph::drawErrorsBefor()
40  or drawErrorsAfter() so it calls the plotErrorIndicators() method from this class. In addition this class
41  will add some public datamemebers and methods to your class that allow to specify the properties of the
42  error indicators (plot properties: color, width, ... and columns for the data).
43 
44  \image html jkqtplotter_simpletest_errorbarstyles.png "line-graphs with different types of error indicators"
