Detailed Description

This implements a single vertical boxplot as a "geometric element", where the data is directly given to the object and not stored in a column, as in JKQTPBoxplotVerticalGraph.

the x position is given in posColumn. All other data are given in the median, min, max, percentile25 and percentile75.
